How to use coordinator?

Coordinator是用来协调各个线程的。
引用tensorflow官网上的说明：

Any of the threads can call coord.request_stop() to ask for all the threads to stop.
To cooperate with the requests,
each thread must check for coord.should_stop() on a regular basis.
coord.should_stop() returns True as soon as coord.request_stop() has been called.

在调用coord.request_stop() 并且等待 enq_threads都终止之后，如果我们继续 一直执行出队操作x,那么当队列空时会产生OutOfRangeError。
